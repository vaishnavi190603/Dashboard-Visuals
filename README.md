HR Analytics Dashboard

This project also includes an interactive HR Analytics Dashboard created using Tableau for data visualization and workforce insights.

The dashboard provides:
- Department wise Attrition
- Education field wise Attrition
- Job Satisfaction Rating
- No. of Employee by Age group
- Attrition by Gender
- Attrition Rate by Gender for different age group
- Real-time HR data visualization

**View Live Dashboard Here:**  
https://public.tableau.com/app/profile/vaishnavi.bhandarkar3651/viz/HRDashboard_17793664487460/HRANALYTICSDASHBOARD

> The dashboard is fully interactive and allows users to explore Job Satisfaction Rating, Department wise Attrition, and analytical insights in real time.
